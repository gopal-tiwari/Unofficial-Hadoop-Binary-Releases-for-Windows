Welcome Windows Geeks!!!.

Hadoop provides native Windows support from version 2.2 and above, but for that we need to build it on our own, as official Apache Hadoop releases do not provide native Windows binaries.

Here you will find all Hadoop binary releases from & onwards version 2.2 for Windows platform. You can download it using the links given in the download section below or alternatively from the Releases section of my origional project repo here.

Compilation Process

For build process, we followed official Hadoop build instructions listed here. For more details on build process please follow the toturial @ dzone.com/articles/build-and-install-hadoop-on-windows-with-native-bi .
